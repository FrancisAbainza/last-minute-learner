# Last Minute Learner

A comprehensive learning platform designed to help you quickly grasp and master new topics.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Technologies Used

-   **Next.js:** A React framework for building full-stack web applications.
-   **Clerk:** For authentication and user management [[memory:4151076]].
-   **Drizzle ORM:** A TypeScript ORM for seamless database interactions [[memory:4316756]].
-   **shadcn/ui:** A collection of re-usable components built using Radix UI and Tailwind CSS [[memory:4317208]].
-   **Tailwind CSS:** A utility-first CSS framework for rapid UI development.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Features

- **Interactive Learning:** Engage with topics through a dynamic and user-friendly interface.
- **Quick Comprehension:** Designed for efficient learning, helping you grasp concepts in minimal time.
- **Personalized Dashboard:** Track your progress and manage your learning journey.
- **Scalable Architecture:** Built with modern technologies for performance and maintainability.
